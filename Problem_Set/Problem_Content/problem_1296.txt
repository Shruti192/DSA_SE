b"\nChef has an array AA which is a permutation of natural numbers till nn. He defines value of an array as no of elements in the array whose neighboring elements have sum at least kk.(Note that first and last element of an array have only 1 neighboring elements where as remaining elements have 2 neighboring elements each.)\nYou don't know the permutation that chef has, so you decide to find the sum of the values of all possible permutations. Given nn and kk you need to find this sum and output it modulo 1000000007 (109+7109+7).\nInput:\n\nFirst line will contain TT, number of testcases. Then the testcases follow. \nEach testcase contains of a single line of input, two integers n,kn,k. \n\nOutput:\nFor each testcase, output in a single line sum of value of all possible permutations modulo 1000000007.\nConstraints\n\n1\xe2\x89\xa4T\xe2\x89\xa41000001\xe2\x89\xa4T\xe2\x89\xa4100000\n2\xe2\x89\xa4n\xe2\x89\xa41052\xe2\x89\xa4n\xe2\x89\xa4105\n2\xe2\x89\xa4k\xe2\x89\xa41052\xe2\x89\xa4k\xe2\x89\xa4105\n\nSubtasks\n\n30 points :\n2\xe2\x89\xa4n\xe2\x89\xa410002\xe2\x89\xa4n\xe2\x89\xa41000\n2\xe2\x89\xa4k\xe2\x89\xa410002\xe2\x89\xa4k\xe2\x89\xa41000\n70 points : Original constraints\n\nSample Input:\n2\n2 2\n3 3\n\nSample Output:\n2\n10\n\nEXPLANATION:\nIn first case since only 2 permutations are possible (1,2) and (2,1) and each of which has only one element 1 whose sum of neighboring elements is 2, value of both permutations is 1 and hence required ans is 2.\nFor 2nd test case, we have following permutations:\n1,2,3 - Value = 1\n1,3,2 - Value = 3\n2,1,3 - Value = 1\n2,3,1 - Value = 3\n3,1,2 - Value = 1\n3,2,1 - Value = 1\n\nHence, ans is 1+3+1+3+1+1 = 10\n\n\n\n\nAuthor:\n7\xe2\x98\x85yash_chandnani\n\n\nEditorial:\nhttps://discuss.codechef.com/problems/PERMSUM\n\n\nTags:\n\n               \n                 finding-pattern, locmay18, math, medium, yash_chandnani\n                 \n               \n             \n             \n\n\n\n                        Difficulty Rating:\n                    \n\nN/A\n\n\nDate Added:\n24-05-2018\n\n\nTime Limit:\n1 secs\n\n\nSource Limit:\n50000 Bytes\n\n\nLanguages:\nPYTH 3.6, JAVA, C, CPP14, PYTH, PYP3, CS2, ADA, PYPY, TEXT, PAS fpc, NODEJS, RUBY, PHP, GO, HASK, TCL, kotlin, PERL, SCALA, LUA, BASH, JS, rust, LISP sbcl, PAS gpc, BF, CLOJ, D, CAML, swift, FORT, ASM, FS, WSPC, LISP clisp, SCM guile, PERL6, ERL, CLPS, PRLG, ICK, NICE, ICON, COB, SCM chicken, PIKE, SCM qobi, ST, NEM\n\n\n\n\n\nSubmit\n\n"
